# OAuth2 Demo (Frontend React)
This project is OAuth2 Demo created with React.

Backend is [here (FastAPI)](https://github.com/t-okuji/fastapi-oauth-demo).

## Implemented

- Google
- Apple

## Getting started

### Install libraries
```
bun install
```

### dev

```
bun run dev
http://localhost:5173
```

### debug

F5 to launch the debugger. 

### build

```
bun run build
```
